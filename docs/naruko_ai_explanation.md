# 鳴子ダム 渇水予測AI（人工知能）の仕組みと予測方法

このドキュメントでは、鳴子ダムの渇水リスク（水不足の危険度）を予測するために導入された最新のAIモデル（v3）の仕組みを解説します。

---

## 1. AIは何を予測しているのか？
このAIは、鳴子ダムの将来の水位を **7日後、28日後（約1ヶ月後）、60日後、90日後（約3ヶ月後）** の4つのスパンで予測します。

*   **短期予測（7日後）**: **OpenWeatherMap API** から取得する高精度な週間予報を直接読み込み、直近の雨や気温の変化が水位にどう響くかを緻密に予測します。
*   **中長期予測（28日〜90日後）**: 季節ごとのダムの運用ルールや、山に積もった「雪の貯金」がいつ溶け出すかといった、長期的なトレンドを予測します。

## 2. AIの「知恵」の源（学習データ）
AIは、**過去約30年間分（1993年〜2023年）の膨大な実績データ** を繰り返し学習し、パターンを暗記しています。

*   **ダムの実績**: 過去30年間の毎日の水位、流入量、放流量。
*   **気象の実績**: 鳴子周辺（川渡アメダス）の気温、降水量、積雪深、降雪量。
*   **運用ルール（月情報）**: 「この時期は田植えのために水を残す」「この時期は洪水に備えて水位を低く保つ」といったダム特有の運用サイクルを月単位で学習しています。

## 3. 「いま」読み込んでいる最新データ
予測を行う瞬間、AIは以下の最新情報を組み合わせて計算します。

1.  **現在のダムの状態**（宮城県のリアルタイムデータ）
    *   今の水位、今の流入量、今の放流量。
2.  **高精度な天気予報**（OpenWeatherMap API）
    *   向こう7日間の **正確な予想降水量 (mm)** と **予想平均気温 (℃)**。これにより、単なる降水確率ではなく「何mmの雨が降るか」を予測の根拠にしています。
3.  **リアルタイムの雪情報**（アメダス統計）
    *   **現在の積雪深 (cm)** と、**最近1週間に降った雪の量 (cm)**。
    *   さらに、**過去30日間の平均積雪深** を算出することで、「山にどれくらいの融雪予備軍（雪の貯金）があるか」を評価し、春先の流入量を予測します。

## 4. 予測の仕組み：ランダムフォレスト
このAIは **「ランダムフォレスト」** という手法を採用しています。

1.  **100人の仮想専門家**: AIの中に、それぞれ異なる視点で過去を学んだ100人の専門家を用意します。
2.  **独自のルールで予測**: 専門家たちは「今の水位がこうで、来週雨が20mm降るなら、過去の経験から7日後はこうなる」と各自で予測を出します。
3.  **多数決と平均**: 100人の予測の平均値を最終的な「予想水位」とし、意見のバラつきを「予測の自信（±表示）」として出力します。

## 5. 予測結果と危険度判定
AIが算出した予想水位に基づき、ダムの有効貯水容量に照らして危険度を判定します。

*   **NORMAL（正常）**: 貯水率に余裕があり、安定した供給が可能です。
*   **CAUTION（注意）**: 水位が低下傾向にあります。今後の気象変化に注意が必要です。
*   **WARNING（警戒）**: 渇水の兆候が鮮明です。節水の検討が必要なレベルです。
*   **CRITICAL（危険）**: 本格的な渇水状態です。取水制限などの厳しい対策が必要になる水位です。

---

**まとめ**
このAIは、**「過去30年の歴史」**＋**「現在のリアルタイムな雪の情報」**＋**「世界基準の高精度な週間天気予報」** を組み合わせることで、人間では判断が難しい「複雑な気象変化とダム運用の相関」を捉えようとしています。
特に短期予測においては、世界トップクラスの気象APIを統合したことで、従来の疑似予報よりもはるかに実用的な精度を実現しています。
