# 宮城県 渇水状況モニター (Miyagi Drought Watch) 全体設計図

## 1. プロジェクト概要
**目的:** 宮城県内（一部国交省管理を含む）の主要ダムの貯水状況をリアルタイムでモニタリングし、渇水リスクを可視化するWebアプリケーション。
**想定ユーザー:** 防災担当者、インフラ管理者、一般県民など。
**主要機能:**
- 地図上でのダム位置と渇水レベル（正常、注意、警戒、危険）のプロット
- ダムのリアルタイム貯水率、流入・放流量などのステータス表示
- 水系および主要河川のネットワーク表示
- 主要河川の水位観測所のリアルタイム水位表示と渇水基準比の評価
- 気象庁API連携による簡易的な天気予報表示

---

## 2. システムアーキテクチャ

システムは **Node.js + Express (バックエンド)** と **Vite + Vanilla JS (フロントエンド)** による構成で動作します。

```text
[ ブラウザ (Client) ]
    │
    ├─ UI描画 (HTML/CSS/JS)
    ├─ 地図レンダリング (Leaflet.js + OSM GeoJSON + マーカー群)
    └─ データ取得 (fetch API)
         │
  (HTTP GET /api/dams, /api/water-levels, /api/weather)
         ▼
[ バックエンドサーバー (Node.js/Express) ]
    │
    ├─ APIルーティング (server/index.js)
    ├─ データ集約層 (DataAggregator.js)
    │    ├─ 渇水判定 (DroughtAnalyzer.js)
    │    └─ キャッシュ管理 (CacheManager.js)
    │
    ├─ モックデータ生成 (mockDams.js / 障害時フォールバック用)
    │
    └─ 外部データ取得サービス
         ├─ ダム現況スクレイパー (MiyagiDamScraper.js) ──▶ [ 宮城県 河川流域情報システム ]
         ├─ 水位観測所スクレイパー (WaterLevelScraper.js) ──▶ [ 国土交通省 水文水質データベース ]
         └─ 天気予報API連携 ──────────────────────────▶ [ 気象庁 防災情報API ]
```

## 4. データ設計・フロー

### 4.1. データマスタ (`server/config/regions/miyagi.js`)
すべての基盤となるダム情報（ID、名称、水系、緯度経度、有効容量、利水容量、国/県管理区分など）を静的オブジェクトとして管理します。外部からデータを取得する際は、このマスタと紐付けて処理を行います。

### 4.2. リアルタイムデータの流れ
1. **スクレイピング:** 
   - `MiyagiDamScraper` が宮城県の公開サーバーからHTMLを取得・パースし、リアルタイムの貯水率や放流量を抽出します。
   - `WaterLevelScraper` が水文水質データベースから10分間隔のリアルタイム水位データをパース抽出します。
2. **分析・判定:** `DroughtAnalyzer` が取得したダム貯水率をもとに、定義されたしきい値（例: >80% 正常, <40% 警戒, <30% 危険）で渇水レベルを判定します。また、水位観測所データは正常流量・渇水目安流量と比較されます。
3. **集約・キャッシュ:** `DataAggregator` およびAPIハンドラーは解析済みデータと天気データを集約し、さらに外部アクセス負荷を減らすため10分間のキャッシュ（`CacheManager`）を挟んでフロントエンドに返却します。
4. **表示:** フロントエンドが `/api/dams` や `/api/water-levels` からJSONを受け取り、地図マーカー（円とダイヤ型）・リスト・サマリーカード・詳細モーダルの各UIにバインドします。

## 5. UI/UX デザインポリシー

- **世界観:** 「サイバー・インフラストラクチャー / 防災ダッシュボード」
- **配色:** 深いスレートグレー（Slate）を基調としたダークモード。重要情報のみにネオンカラー（シアン、ブルー、イエロー、レッド等）を使用し視認性を向上。
- **インタラクション:** 
  - 地図上のダム（光るドット）をクリックするとPopupと左サイドパネルが連動。
  - リスト等にホバーアニメーションやGlassmorphism（すりガラス効果）を採用し、モダンで直感的な操作感を実現。
- **空間情報の充実:** CARTO Dark Matterタイルに加え、主要河川ネットワークを青線で補完描画し、「水系」を通じたダムと地域の繋がりを視覚的に表現。
